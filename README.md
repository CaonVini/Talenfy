# Talenfy

Fiz essa ferramenta pra resolver um problema que eu tinha: queria saber se meu currículo tava bom pra uma vaga antes de aplicar.

Funciona assim: você cola a descrição da vaga, cola seu currículo (ou sobe o PDF), e recebe uma análise mostrando o quanto você combina com a vaga. Além do score, mostra seus pontos fortes, o que tá faltando, e dicas práticas pra melhorar.

A análise é feita pelo Gemini, então você precisa de uma chave de API. É grátis - só pegar no [Google AI Studio](https://aistudio.google.com/apikey) e colar nas configurações (ícone de engrenagem ⚙️).

Sua chave fica salva só no seu navegador, nada vai pro servidor.
